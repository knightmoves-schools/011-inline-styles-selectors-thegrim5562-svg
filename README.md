# 011 Inline Style And Selectors

[![011 Inline Style And Selectors](https://img.youtube.com/vi/0PLGlOIpA9M/0.jpg)](https://www.youtube.com/watch?v=0PLGlOIpA9M)

The `index.html` file
- should use an `inline` style to style the `span` to be `orange`
- should use an `element selector` to style the `<h1>` element to be grey
- should use a `class selector` to style the element with the `brown-text` class to be brown
- should use an `id selector` to style the element with the `selector-id` id to be pink

Resources:
- https://www.w3schools.com/cssref/css_selectors.asp
- https://www.w3schools.com/cssref/pr_text_color.asp
